## usage
```sh
node lib/index.js --name "Nice Movie" --hl "KILLERS" "FUM"
```

The keywords that you pass, are used to figure the best match between Movie/Subtitle.

## pre-requisites

You must have VLC installed and have a Node version of 6+.
